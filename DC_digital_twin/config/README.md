# Конфигурация

Файл `config.yaml` **удалён**. Источник настроек по умолчанию — модуль `src/default_config.py`.

- **Веб**: страница **«Конфигурация»** (`/config`) — форма или JSON, сохранение через API.
- **API**: `GET /config`, `PUT /config`, `GET /config/defaults`.
- **CLI**: `python main.py` без `--config` использует тот же встроенный конфиг.
- **Опционально**: переменная окружения `CONFIG_PATH` — путь к YAML-файлу; если файл существует, он используется при старте API.
