## Интерпретация

- **со спадом** — траектория T_room не только росла: было снижение (работает отбор тепла).
- При **free** и жаркой наружке после фикса ожидается доля со спадом > 0 и разумный ΔT.

### Подробный разбор

1. **Sanity-check** перед серией: `free` + `T_out=35°C`, 50 шагов с `dt=10` с → если `T_room < 23°C`, процесс API загрузил актуальный `cooling.py` (механика к уставке при `T_out ≥ T_ret`). Если `T_room > 23°C` — **перезапустите uvicorn** из каталога проекта с `PYTHONPATH=.` (старый процесс давал `Q=0` и рост зала).

2. **ΔT_mean** — среднее по 25 сидам изменение `T_room` от первого до последнего сэмпла (20 точек по траектории). Отрицательное значение — зал в среднем **охлаждается** за прогон.

3. **free_Tout24** — чуть больший `final_T_std` и выше `final_T_mean`, чем при 35°C: при умеренной наружке баланс (стены + экономайзер) меняется, плюс случайная нагрузка даёт разброс между сидами.

4. **free_Tout35** vs **mixed_Tout35** vs **chiller_Tout35** — при данных параметрах траектории совпали (одинаковые `final_T_mean` / `ΔT`): для воздушной стороны до уставки даётся один и тот же `Q_act`; различия сильнее в **PUE** (учёт компрессора в `compute_total_power`).

5. **free_Tout18** — самый сильный средний охлаждающий эффект (ΔT_mean ≈ −6.3°C): холодная наружка усиливает отвод через ограждение и режим free.
